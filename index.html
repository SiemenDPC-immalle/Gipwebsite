FRANS
Reportage photo du stage
In deze taak voor Frans moesten we eerst fotos nemen in ons bedrijf van verschillende kamers en later in het Frans beschrijven. Het doel van deze opdracht was om onze woordenschat in de werkomgeving te verbeteren.
Ik heb deze taak gemaakt in Prezi, een gemakkelijke website waar je eigen gegevens visueel kan maken. Deze opdracht vond ik best leuk, daar het ons aanmoedigde om foto's te nemen tijdens de stage, wat me enorm heeft geholpen voor de GIP website te decoreren. Op de achtergrond kunt u een voorbeeld zien van "Photo 2", hiervan zijn er nog 7 andere.

CASE STUDY
Case-studies in stage-bedrijf
Azure DevOps: Een combinatie van ontwikkeling (Dev) en bedrijfsactiviteiten (Ops), is de samenstelling van mensen, processen en technologie om doorlopend waarde aan klanten te bieden.
Een paar functies die deze applicatie kan doen is bijvoorbeeld, Work item = staat de uitleg wat je moet doen, naam, geschiedenis, descriptie
Pipelines = code samenpakken en configureren. Wanneer de code goedgekeurd wordt, create dev ops een app die alleen code neemt van de master, maar de code zelf ook nog eens controleert en het vergelijkt met de vorige versie van de app, om te kijken of er geen breaking changes zijn.

ERD-model
We hebben ons ERD-model in LucidChart gemaakt. Met dit programma kan je snel ERD-modellen maken die duidelijk en overzichtelijk zijn.
In dit ERD-model focus ik op de tblKlanten. Deze tabel maakt een ticket aan in tblTickets en behoort tot tblBedrijf. In de tblKlanten kan je de gegevens van de klanten opslagen en gebruiken.

PROGRAMMEREN
GIP Website
Het programmeren van deze site was een leuke uitdaging waarmee ik veel heb geleerd over HTML, CSS en zelfs JavaScript.
De grote uitdaging die ik had met mijn website was het gebruiken van een Template. Het begrijpen en lezen van de template was soms moeilijk sinds er JavaScript bij zit en de CSS is op de instellingen van de Template waardoor je goed moet opletten welke klassen je moet gebruiken.
Waarom heb ik deze template dan gekozen? Ik hou van de simpele UI en de foto die ik nu gebruik in de achtergrond inplaats van naast de uitleg maakt het geheel vernieuwend.
