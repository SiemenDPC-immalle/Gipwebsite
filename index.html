CASE STUDY
Case-studies in stage-bedrijf
Azure DevOps Een combinatie van ontwikkeling (Dev) en bedrijfsactiviteiten (Ops), is de samenstelling van mensen, processen en technologie om doorlopend waarde aan klanten te bieden. Het helpt dus vooral om bedrijven samen te zetten en waar je duidelijk kunt zien wie wat aan het doen is, met dit voorbeeld kan je zien dat het principe niet echt moeilijk is, je kunt zien wie wat aan het doen is, wat er moet gebeuren en wat er al klaar is.
Een paar functies die deze applicatie kan doen is bijvoorbeeld, Work item = staat de uitleg wat je moet doen, naam, geschiedenis, descriptie Pipelines = code samenpakken en de configureren Wanneer de code goedgekeurd word, create dev ops een app die alleen code pakt van de master, maar de code zelf ook nog eens controleert en het vergelijkt met de vorige versie van de app, om te kijken of er geen breaking changes zijn.
